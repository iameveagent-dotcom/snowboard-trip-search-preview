## Variant: Map-first explorer

### Design stance
Geographic exploration leads; cost estimates appear as regional overlays.

### Key choices
- Layout: side decision panel plus full map canvas.
- Typography/color: Linear dark with luminous route overlays.
- Interaction: selectable region options.

### Trade-offs
- Strong at: understanding airports/resort geography.
- Weak at: less immediate detailed cost breakdown.

### Best for
Users choosing between unfamiliar regions or international trips.
