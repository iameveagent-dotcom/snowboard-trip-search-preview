## Variant: Dashboard command center

### Design stance
Dense Linear-style command center for acting on a flight sale quickly.

### Key choices
- Layout: search bar, recommendation rail, map, estimate breakdown.
- Typography: Inter/JetBrains Mono, dark precision UI.
- Interaction: Estimate button highlights the total cost panel.

### Trade-offs
- Strong at: fast decisions, cost visibility.
- Weak at: less emotional/travel editorial feel.

### Best for
Power users comparing airfare windows under time pressure.
